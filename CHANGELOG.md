# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added
- Initial release
- Core `<Pip>` component with draggable UI PiP functionality
- `<PipContainer>` component for portal targets
- Controlled and uncontrolled modes
- Window-bounded dragging
- Snap-to-corners feature
- Position persistence via localStorage
- Custom drag handle support
- Full TypeScript support
- SSR-safe for Next.js (App Router & Pages Router)
- Zero external dependencies
- Comprehensive documentation and examples
